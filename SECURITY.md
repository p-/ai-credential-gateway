# Security Policy

## Supported Versions

Only the latest released (non-experimental) version is supported. At this time no version has been released. 

## Reporting a Vulnerability

Use GitHub's [private vulnerability reporting](https://github.com/p-/ai-credential-gateway/security/advisories/new) feature to report a vulnerability.

## Security Considerations.

* This gateway is intended to be used locally or in controlled environments and should NEVER be exposed to uncontrolled or even public networks.
